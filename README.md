# Hyperion AI
Autonomous experimentally trained AI

## Features

- **Voice-Enabled Interface**: Powered by ElevenLabs text-to-speech API
- **Autonomous Intelligence**: Experimentally trained AI system
- **Cross-Platform**: Web and mobile applications

## ElevenLabs Integration

Hyperion AI now features voice capabilities using the ElevenLabs API. See [ElevenLabs Integration Guide](docs/ELEVENLABS_INTEGRATION.md) for setup instructions.

### Quick Start

1. Get your ElevenLabs API key from [elevenlabs.io](https://elevenlabs.io/)
2. Configure the API key in `docs/hyperion-prompt.html`
3. Open the interface and start issuing voice commands

For detailed setup and usage instructions, see the [full integration guide](docs/ELEVENLABS_INTEGRATION.md).
