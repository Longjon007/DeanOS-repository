# Agent Training Exercises

## Objective

Train the agent to identify bugs, improve test coverage, and optimize code.

## Exercises

1. **Test Coverage**:
   - Create tests for `web` components and utilities.
   - Create tests for `app` screens and utilities.
   - Create tests for `supabase` edge functions.
   - Run coverage reports and identify gaps.

2. **Refactoring**:
   - Identify code duplication and refactor into reusable functions.
   - Ensure proper typing (TypeScript) where applicable.
